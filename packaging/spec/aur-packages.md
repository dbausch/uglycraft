## Purpose

Make both games installable on Arch Linux via AUR. One PKGBUILD produces two
independently installable packages so users can choose either or both.

## What ships

### Package `uglycraft`

The Python/pygame spiritual remake. Installed as a set of Python source files
with system package dependencies.

| Installed path | Contents |
|---|---|
| `/usr/share/uglycraft/*.py` | `main.py`, `game.py`, `constants.py`, `sprites.py`, `levels.py`, `entities.py`, `hiscore.py`, `sounds.py` |
| `/usr/share/uglycraft/fonts/ShareTechMono-Regular.ttf` | Bundled font |
| `/usr/bin/uglycraft` | Wrapper: `exec python /usr/share/uglycraft/main.py "$@"` |
| `/usr/share/licenses/uglycraft/GPL-3.0.txt` | Game license |
| `/usr/share/licenses/uglycraft/OFL-1.1-ShareTechMono.txt` | Font license |
| `/usr/share/icons/hicolor/scalable/apps/uglycraft.svg` | Application icon |
| `/usr/share/applications/uglycraft.desktop` | Freedesktop launcher |

**depends**: `python`, `python-pygame`, `python-numpy`

Font path resolution: `game.py` uses `os.path.dirname(os.path.abspath(__file__))`
as base directory, so the font is found at
`/usr/share/uglycraft/fonts/ShareTechMono-Regular.ttf` without any code changes.

High-score file: `constants.py` already uses `$XDG_DATA_HOME/uglycraft/uglycraft.hsc`.

### Package `ugli`

The 1996 DOS text-mode game, compiled with Free Pascal for Linux.

| Installed path | Contents |
|---|---|
| `/usr/share/ugli/UGLI_2` | FPC-compiled ELF binary |
| `/usr/share/ugli/translations/*.mo` | Compiled gettext translations (German) |
| `/usr/share/ugli/ANSI-87.conf` | Kitty colour theme for accurate rendering |
| `/usr/bin/ugli` | Bash wrapper (see below) |
| `/usr/share/licenses/ugli/GPL-3.0.txt` | Game license |
| `/usr/share/icons/hicolor/scalable/apps/ugli.svg` | Application icon |
| `/usr/share/applications/ugli.desktop` | Freedesktop launcher |

**optdepends**:
- `portaudio` — sound support via UOS/PortAudio
- `kitty` — terminal launcher with ANSI-87 colour theme
- `ttf-liberation` — Liberation Mono font for kitty launcher

Translation loading: `LoadTranslation` in `UGLI_2_Core.inc` uses `ParamStr(0)`
to find the binary directory, then probes `<exedir>/translations/%s.mo`. Since
the wrapper invokes `/usr/share/ugli/UGLI_2` by absolute path, translations
resolve to `/usr/share/ugli/translations/de.mo`.

High-score file: UGLI writes `UGLI.HSC` to cwd. The wrapper cd's to
`${XDG_DATA_HOME:-~/.local/share}/ugli/` before launching.

## `packaging/ugli.sh` — wrapper script

Mirrors `poe run-original` with a fallback chain:

```bash
#!/bin/bash
UGLI=/usr/share/ugli/UGLI_2
DATA_DIR="${XDG_DATA_HOME:-$HOME/.local/share}/ugli"
mkdir -p "$DATA_DIR"
cd "$DATA_DIR"

for arg in "$@"; do
  case "$arg" in -h|--help)
    exec "$UGLI" "$@" ;; esac
done

if [ -n "$TERMINAL" ]; then
  exec $TERMINAL "$UGLI" "$@"
elif command -v kitty >/dev/null 2>&1; then
  exec kitty \
    -c /usr/share/ugli/ANSI-87.conf \
    -o font_family="Liberation Mono" \
    -o font_size=16 \
    -o remember_window_size=false \
    -o initial_window_height=25c \
    -o initial_window_width=80c \
    -- "$UGLI" "$@"
else
  exec "$UGLI" "$@"
fi
```

1. Creates `$XDG_DATA_HOME/ugli/` and cd's there (for `UGLI.HSC`)
2. `--help` / `-h` → exec binary directly (outputs to current terminal)
3. `$TERMINAL` set → use it
4. kitty available → open new window with ANSI-87 theme, 80x25, Liberation Mono
5. Fallback → exec binary directly in current terminal

## Icons

### `packaging/uglycraft.svg`

SVG rendering of the player sprite from `sprites.py:draw_player()`:

- Gold filled circle (rgb 255, 215, 0) with darker outline (rgb 180, 150, 0)
- Two dark eyes in the upper third, each with a white highlight dot
- Curved smile below centre
- Bold and simple — must read clearly at 48×48 and 16×16 in a taskbar

Reference: the pygame code draws a circle at `(cx, cy)` radius `size/2 - 3`,
eyes at `cy - r/3` spaced `r/3` from centre, smile as a cos/sin arc below
centre.

### `packaging/ugli.svg`

Bold block-letter rendering of **UGLI**, inspired by the intro ASCII art.
The original ASCII art uses `**` characters to form each letter:

```
**        **    **********   **           **
**        **   **        **  **           **
**        **   **        **  **           **
**        **   **            **           **
**        **   **     *****  **           **
**        **   **        **  **           **
**        **   **        **  **           **
 **********     **********   **********   **
```

The SVG condenses this into a compact, bold blocky glyph — thick strokes,
tight letter-spacing, high contrast. Must be clearly legible at small sizes
(16×16 taskbar). Use a single strong colour (e.g. red or white on
transparent) rather than multi-colour.

## Desktop files

### `packaging/uglycraft.desktop`

```ini
[Desktop Entry]
Type=Application
Name=UGLYCRAFT
GenericName=Arcade Game
Comment=Treasure-collecting arcade game inspired by UGLI (1996)
Exec=uglycraft
Icon=uglycraft
Terminal=false
Categories=Game;ArcadeGame;
Keywords=game;arcade;retro;
```

`Terminal=false` — UGLYCRAFT opens its own pygame/SDL window.

### `packaging/ugli.desktop`

```ini
[Desktop Entry]
Type=Application
Name=UGLI
GenericName=Text-Mode Game
Comment=UGLI 2 (1996) — text-mode treasure game (Linux/FPC port)
Exec=ugli
Icon=ugli
Terminal=false
Categories=Game;ArcadeGame;
Keywords=game;arcade;retro;terminal;
```

`Terminal=false` — the `ugli` wrapper opens its own kitty window (or uses
`$TERMINAL`). It does not need the desktop environment to supply a terminal.

## PKGBUILD structure

### Sources

```
source=("$pkgbase-$pkgver.tar.gz::$url/archive/v$pkgver.tar.gz"
        uos.pas::https://raw.githubusercontent.com/fredvs/uos/main/src/uos.pas
        uos_flat.pas::https://raw.githubusercontent.com/fredvs/uos/main/src/uos_flat.pas
        uos_portaudio.pas::https://raw.githubusercontent.com/fredvs/uos/main/src/uos_portaudio.pas
        ANSI-87.conf::https://raw.githubusercontent.com/kovidgoyal/kitty-themes/master/themes/ANSI-87.conf)
```

UOS and ANSI-87 are declared as explicit sources rather than fetched by curl
during build (matching AUR conventions).

**makedepends**: `fpc`

### prepare()

Copy UOS sources and ANSI-87.conf into the source tree:

```bash
mkdir -p original/uos
cp "$srcdir"/{uos.pas,uos_flat.pas,uos_portaudio.pas} original/uos/
cp "$srcdir/ANSI-87.conf" original/
```

### build()

Only the Pascal game needs compilation:

```bash
cd original && fpc -Fuuos UGLI_2.pp
```

### package_uglycraft() / package_ugli()

Install files as listed in the tables above using standard `install -Dm{644,755}`.
